# 🧩 패턴 퍼즐 게임

보드게임을 웹으로 구현한 패턴 인식 퍼즐 게임입니다.

## 🎮 게임 방법

1. **"새 문제" 버튼** 클릭하여 퍼즐 시작
2. 주어진 패턴의 **규칙을 찾기**
3. 빈 칸에 **알맞은 모양과 색상** 그리기

## 🛠️ 도구

### 모양 도장 (5가지)
- 🔵 원
- ⬛ 사각형
- 🔺 삼각형
- ⭐ 별
- ❤️ 하트

### 그리기 도구
- ✏️ **연필**: 자유롭게 선 그리기
- 🧹 **지우개**: 그린 내용 지우기

### 색상 (3가지)
- ⚫ 검정
- 🔴 빨강
- 🔵 파랑

## 📋 사용 방법

### 도장 찍기
1. 원하는 모양 버튼 클릭
2. 보드에서 원하는 위치 클릭
3. 해당 위치에 모양이 찍힘

### 선 그리기
1. "연필" 도구 선택
2. 원하는 색상 선택
3. 보드에서 마우스 드래그

### 지우기
1. "지우개" 도구 선택
2. 지우고 싶은 부분 드래그

## 🎯 패턴 예시

### 색상 패턴
```
빨강 - 파랑 - 검정 - 빨강 - ...
```

### 모양 패턴
```
원 - 사각형 - 삼각형 - 별 - 하트 - 원 - ...
```

### 혼합 패턴
```
행별로 다른 색상
열별로 다른 모양
```

## 🚀 실행 방법

웹 브라우저에서 `index.html` 파일을 열면 됩니다.

```bash
# 간단한 로컬 서버로 실행 (선택사항)
python -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## ✨ 기능

- ✅ 5가지 모양 도장
- ✅ 3가지 색상 (빨강, 파랑, 검정)
- ✅ 연필 그리기 도구
- ✅ 지우개
- ✅ 전체 지우기
- ✅ 힌트 시스템
- ✅ 랜덤 퍼즐 생성
- ✅ 터치 디바이스 지원
- ✅ 반응형 디자인
